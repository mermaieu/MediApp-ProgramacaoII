Nessa pasta ficam os controladores da aplicação.

Num controller, temos ações de mapeamento de rotas.

Cada controlador precisa ser preparado para ser reconhecido como um módulo e 
responder às requisições adequadamente. Para isso, cada arquivo de controlador 
deve importar o Express e utilizar o método router para definir as rotas 
específicas daquela parte da aplicação.

O arquivo router.js contém as rotas de mapeamento da aplicação.
É o roteador.
